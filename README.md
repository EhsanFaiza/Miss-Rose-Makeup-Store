# Miss-Rose-Makeup-Store

## Overview

Miss Rose Makeup Store is a web-based e-commerce application developed as a university software project. It allows customers to browse cosmetics and beauty products, search by category or brand, manage a shopping cart, and place orders through a user-friendly interface.

This project was developed using Object-Oriented PHP with MySQL as the database and Bootstrap for a responsive user interface.

---

## Features

- User registration and login
- Browse products by category
- Search and filter products
- Product details page
- Shopping cart
- Order placement
- Order history
- Cancel pending orders
- Admin dashboard
- PayPal payment integration

## Installation

1. Install XAMPP.
2. Copy the project folder into the `htdocs` directory.
3. Start Apache and MySQL.
4. Import the SQL database into phpMyAdmin.
5. Update database credentials if required.
6. Open the project in your browser:

```
http://localhost/MissRoseMakeupStore
```
